# Roadmap

## 目的

本文档用于记录《射电干涉孔径合成基础》中文项目在“当前版本已经完成主体中文化和风格统一”之后，下一阶段值得推进的补强方向。

当前判断是：

- 本书已经具备较完整的干涉测量理论主线，适合作为中文原理教材。
- 如果希望进一步对标近年的国际射电干涉训练体系，则主要缺口已不在基础理论，而在现代数据处理工作流、专题实践和可复现实操材料。

近期工作原则：

- 暂时不开新章节，不先扩展第 10 章及以后内容。
- 优先按现有第 1 至第 9 章逐章回看，把已经存在的内容讲清、讲明、讲厚。
- 每一章的补强都应服务现有主线，避免把章节变成新专题标题的堆积。


## 现状概括

当前仓库已经较完整覆盖以下内容：

- 射电科学背景与基础天体物理
- 数学基础、傅里叶分析、采样、卷积、最小二乘
- 位置天文学与坐标系统
- 可见度空间、基线、UV 覆盖、van Cittert-Zernike 定理
- 成像、PSF、网格化、权重、宽场效应
- 去卷积、CLEAN、残差与图像质量
- 观测系统、Jones 符号、RIME、主波束、极化、传播效应、RFI
- 校准主线，包括 1GC、2GC、3GC

当前最明显的不足集中在以下几类：

- 实践章节数量偏少，尚未形成完整的现代 reduction workflow
- 高级谱线工作流已经建立第一版入口，但还需要继续加厚到更真实的 3D 建模与不确定度传播
- 单碟与干涉阵联合成像、短间距补偿已经建立第一版入口，但还需要继续加深
- 偏振数据处理实战已经建立基础与高级入口，但还需要继续加深到更真实的校准和 Faraday 建模
- 缺少 VLBI 独立处理路径
- 面向现代软件流程的宽带/宽场高级成像实践已经建立基础与方向相关入口，但还需要继续做深
- 缺少从观测设计、归档数据到科学产出的完整流程说明


## 优先级划分

### P0：最值得优先补强

这些内容对全书“从原理教材走向现代数据处理教程”最关键，建议优先推进。

1. 完整数据处理工作流实践

- 目标：给出一条从原始测量集到最终图像的完整路径。
- 推荐内容：
  - 数据检查
  - flagging 思路与常见质量问题
  - bandpass / gain / flux 校准
  - applycal
  - dirty image 与 clean image 对比
  - image assessment 与基本 QA
- 价值：这是国际暑校和 workshop 中最常见、也最实用的训练入口。

2. 谱线数据处理专题

- 目标：补齐 continuum 之外最重要的一类科学处理链路。
- 推荐内容：
  - 频谱窗口与谱线数据结构
  - continuum subtraction
  - cube imaging
  - channel map
  - moment map
  - `3D mask` / source finding / 组件目录
  - PV 图与简化运动学建模
- 价值：目前全书已经建立了谱线基础处理与高级分析入口，下一步应继续加厚到更真实的 3D 运动学与物理量分析链。

3. 数据合并与短间距问题

- 目标：补上现代阵列处理中非常关键的一块。
- 推荐内容：
  - 最大可恢复尺度
  - 为什么会缺短间距
  - interferometer + single-dish 的互补关系
  - feather 思路
  - TP / ACA / 12m 联合成像的概念与案例
- 价值：这是 ALMA 体系和现代毫米波数据处理中非常核心的话题，现已建立第一版入口，下一步应继续加深到更真实的联合处理工作流。

4. 偏振数据处理实战

- 目标：把现有偏振理论与实际处理流程连接起来。
- 推荐内容：
  - Stokes I/Q/U/V 成像
  - leakage 与 D-term 的概念
  - 偏振校准流程概览
  - 极化强度与极化角
  - Faraday rotation / RM 的入门说明
- 价值：当前理论部分已经具备基础，现已建立从入门到 `RM synthesis` 的第一版入口，下一步应继续加深到更真实的校准与 Faraday 建模链。

5. 宽带与宽场高级成像实践

- 目标：让第 5、7、8 章的理论内容落到现代软件方法。
- 推荐内容：
  - multi-frequency synthesis
  - multi-term MFS
  - primary beam correction
  - A-projection / AW-projection 的概念介绍
  - wide-field imaging 的典型参数选择
- 价值：当前已有原理，也已经建立从宽带/宽场入口到方向相关成像的第一版实践链，但还需要继续加深到更真实的软件工作流。

## 编辑策略建议

为了避免未来扩展时再次出现风格不统一，建议新增内容遵守以下原则：

- 继续保持“中文正文为主，英文仅保留软件命令、变量名、专有缩写、论文标题”
- 每个实践 notebook 都尽量明确：
  - 学习目标
  - 输入数据
  - 输出结果
  - 常见问题
  - 延伸阅读
- 对参数选择类内容，优先采用“对比实验 + 结论总结”写法，而不是纯命令堆砌
- 对不同软件任务名，优先解释“它解决什么问题”，再给命令示例
- 保留原理与操作之间的双向链接，例如从第 5、7、8 章跳转到实践 notebook
- 程序、脚本、notebook 和文档中禁止写死个人机器上的绝对路径，例如 `/home/wangfeng/...`
- 一律优先使用相对路径、仓库根目录相对路径，或可配置路径变量，避免内容迁移到其他机器后失效


## 未来新增章节的最低内容标准

后续如果把某个主题单独立成一章，默认不应只保留标题，而应至少包含下面这些部分：

- 背景与科学动机：这一章解决什么问题，为什么需要单独成章
- 核心原理：必要的物理图像、关键方程和变量定义
- 工程或算法流程：对应到真实数据处理时该怎么做
- 自包含示例：至少一个可以运行的 notebook 或等价演示
- 常见失败模式：哪些地方最容易出错，错了会看到什么现象
- 练习或诊断题：让读者真的动手判断一次
- 与前后章节的衔接：这一章读完之后应该接到哪里

如果某个主题暂时还不够厚，就先放在现有章节的加厚部分、延伸阅读或附录里，不急着单独立章。


## 拟新增章节路线图

下面是建议按顺序推进的后续章节。原则是一次只做一章，先做厚，再往前走。

### 第 10 章：VLBI 与高精度干涉

- 背景：为什么 VLBI 不能直接照搬常规综合孔径的处理思路
- 原理：几何时延、fringe phase、delay/rate、coherence loss、closure 量
- 工程流程：记录、相关、fringe fitting、phase referencing、成像与自校准
- 科学出口：高精度天体测量、结构分辨、毫米波 VLBI、e-VLBI
- 建议拆分：至少 3 到 4 个 notebook，分别讲基础、时延/条纹、相位参考、成像/天体测量

### 第 11 章：观测设计、归档数据与再处理

- 背景：从 proposal 到 science-ready data，中间到底发生了什么
- 原理：灵敏度、uv 覆盖、角分辨率、最大可恢复尺度、校准源选择
- 工程流程：scheduling block、archive 下载、pipeline 产品、QA 报告、重处理判断
- 科学出口：如何判断一份公开数据值不值得重跑，如何从 archive 直接进入科学分析
- 建议拆分：至少 3 个 notebook，覆盖观测设计、归档数据解剖、再处理与 QA

### 第 12 章：低频与方向相关校准

- 背景：低频和大视场为什么会让“一个全场增益”失效
- 原理：电离层 TEC、Faraday rotation、station beam、isoplanatic patch、DDE
- 工程流程：direction-independent 到 direction-dependent 的升级路径，peeling、facet、in-field calibrator
- 科学出口：LOFAR / SKA 这类低频与大视场数据的实际处理框架
- 建议拆分：至少 3 个 notebook，分别讲传播误差、方向相关模型和一个最小工作流

### 第 13 章：完整偏振校准与 Faraday 结构

- 背景：为什么偏振不是“把 Q/U/V 画出来”就结束
- 原理：完整 Jones 链、cross-hand delay/phase、D-term、EVPA 标定、去极化
- 工程流程：线/圆馈源差异、偏振校准链、RM synthesis、RM-CLEAN、QU-fitting
- 科学出口：磁场结构、Faraday thick/thin 结构、宽带偏振科学
- 建议拆分：至少 3 个 notebook，覆盖校准链、Faraday 分析、复杂分量诊断

### 第 14 章：谱线三维建模与误差传播

- 背景：moment map 之后，真正的物理分析通常还没开始
- 原理：velocity convention、continuum subtraction、beam smearing、tilted-ring、非圆周运动
- 工程流程：从 cube 到 PV diagram，再到旋转曲线、质量估计和模型比较
- 科学出口：H I、CO、外流、盘结构、动力学质量
- 建议拆分：至少 3 个 notebook，覆盖 3D 掩模、运动学拟合和不确定度传播

### 第 15 章：误差预算、图像保真度与科学量测

- 背景：为什么动态范围高不等于结果可信
- 原理：热噪声、校准误差、去卷积误差、主波束误差、flagging 影响
- 工程流程：flux、size、spectral index、RM、H I mass 等量的误差条如何写
- 科学出口：把图像结果变成可以在论文里汇报的科学量
- 建议拆分：至少 2 个 notebook，一个讲误差预算，一个讲科学量测

### 第 16 章：软件生态与可复现处理

- 背景：不同软件各自擅长什么，为什么不能只会一种工具
- 原理：CASA、WSClean、DP3、DDFacet、killMS、CARTA 等在流程中的位置
- 工程流程：相同问题在不同软件里的对应任务名和数据流转
- 科学出口：帮助读者建立工具选择感，而不是把命令当成知识本身
- 建议拆分：可先作为附录，成熟后再考虑独立成章


## 一句话总结

本书下一阶段最值得做的，不是继续补很多基础理论，而是把现有理论体系延伸成一套面向现代射电干涉数据处理的中文实践教程。
